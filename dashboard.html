How This System Works

  1. User submits credentials → Backend validates and returns a JWT token
  2. Token is stored in localStorage by AuthService
  3. HTTP Interceptor automatically attaches the token to every /api/* request
  4. Backend verifies the token on protected routes using jwt.verify()
  5. On logout, token is removed from localStorage and user is redirected to login
